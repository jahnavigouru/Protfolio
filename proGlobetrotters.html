    Scheduling Assistant

  • This provides one single view with all the scheduled releases of the products (ACC / ACS ) in the form of a calendar.
  • This information helps in effectively scheduling releases.
  • Color coding is used to indicate if the slots are exceeding the product's execution capacity so that the user stays warned while scheduling a new release.
  • Provision to show change restricted slots to align with Adobe wide change restrictions as well as internal UCO specific releases.
